Finn-Signal

Signal console

A personal daily signal engine for Gmail newsletters. Set it up once, then tune it from here.

How it actually runs

Finn-Signal needs one always-awake runner.

Best for family testing Run it on your Mac. The Mac must be awake at delivery time, though Power Nap / a desktop Mac is usually fine.
Best end-state Host it on Render with a persistent disk. Then their computers can be off; Render does the daily work.
What they do Open the page, create profile, connect Gmail, approve newsletters, choose delivery time.
